Mini Browser Mirror 지원

선택한 창을 작은 플로팅 미니 창으로 항상 위에 띄우는 macOS 유틸리티 · macOS 14 이상

한 줄 소개: 메뉴바에서 창 하나를 고르면, 그 화면이 화면 구석의 작은 창에 실시간으로 떠서 항상 위에 보입니다. 전체화면으로 다른 작업을 하면서도 골라둔 창을 계속 지켜볼 수 있어요.

시작하기

  1. 앱을 실행하면 메뉴바에 미니 창 아이콘이 생깁니다.
  2. 처음 사용 시 화면 기록 권한을 요청합니다 — 시스템 설정에서 허용해 주세요.
  3. 메뉴바 아이콘 → 소스 선택 → 미러링할 창을 고릅니다(앱별로 묶여 있습니다).
  4. 화면 구석에 미니 창이 떠서 그 창을 실시간으로 비춥니다.

사용법

  • 켜기/끄기 · 위치(네 모서리) · 크게/작게 — 메뉴바 메뉴에서 조절
  • 미니 창은 마우스로 드래그해 자유롭게 옮길 수 있습니다.
  • 선택한 창은 다음 실행 때 자동 복원됩니다.

자주 묻는 질문

미니 창에 아무것도 안 보여요.

시스템 설정 → 개인정보 보호 및 보안 → 화면 기록에서 Mini Browser Mirror가 켜져 있는지 확인하세요. 권한을 켠 뒤에는 앱을 종료했다 다시 실행해야 적용됩니다.

선택한 창이 목록에서 사라졌어요.

해당 창이 닫히면 미니 창이 잠시 비워지고, 창이 다시 열리면 자동으로 다시 잡습니다. 메뉴를 다시 열면 현재 열린 창 목록으로 갱신됩니다.

녹화도 되나요?

아니요. 이 앱은 화면을 표시만 하며 저장·녹화·전송 기능이 없습니다. 자세한 내용은 개인정보 처리방침을 참고하세요.
