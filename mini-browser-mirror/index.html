Mini Browser Mirror 개인정보 처리방침

최종 개정일: 2026-06-09 · 시행일: 2026-06-09

요약: 이 앱은 사용자 데이터를 수집하지 않습니다. 사용자가 선택한 창의 화면은 같은 Mac 안에서만 작은 플로팅 창으로 표시되며, 어디에도 저장되거나 외부로 전송되지 않습니다. 네트워크 통신·계정·추적이 전혀 없습니다.

1. 수집하는 개인정보

개발자는 어떠한 개인정보도 수집·저장하지 않습니다. 본 앱에는 회원가입·로그인·계정이 없으며, 서버로 전송되는 데이터가 없습니다.

2. 화면 기록 권한

본 앱은 사용자가 직접 선택한 창 하나를 작은 미니 창에 실시간으로 비추기 위해 macOS의 화면 기록 권한을 사용합니다. 캡처된 화면 프레임은 다음과 같이 처리됩니다.

  • 같은 Mac의 화면에 실시간 표시만 하기 위해 메모리에서 처리됩니다.
  • 디스크에 저장하지 않으며, 외부(개발자 서버 포함)로 전송하지 않습니다.
  • 녹화 기능이 없으며, 프레임은 화면에 그려진 즉시 폐기됩니다.

즉, 사용자는 자신의 Mac에 떠 있는 창을 본인이 다시 보는 것이며, 그 내용은 기기를 벗어나지 않습니다.

3. 기기 로컬 저장 항목

아래 설정값만 사용자 기기(macOS UserDefaults)에 저장되며 외부로 전송되지 않습니다.

  • 마지막으로 선택한 소스: 다음 실행 시 같은 창을 복원하기 위한 앱 식별자·창 제목.
  • 미니 창의 표시 관련 설정(위치·크기 등) 등 앱 환경설정.

이 값들은 개인 식별 목적이 아니며 기기 내에만 존재합니다.

4. 제3자 SDK 및 추적

  • 분석·광고 SDK를 사용하지 않습니다.
  • 광고 식별자(IDFA)·SKAdNetwork를 사용하지 않으며, 어떠한 추적도 수행하지 않습니다.
  • 네트워크 통신 자체가 없습니다.

5. 권한

  • 화면 기록 — 사용자가 선택한 창을 캡처하여 미니 창에 표시

해당 권한은 시스템 설정 → 개인정보 보호 및 보안 → 화면 기록에서 언제든 철회할 수 있습니다.

6. 아동의 개인정보

본 앱은 아동을 대상으로 하지 않으며, 아동으로부터 어떠한 정보도 수집하지 않습니다.

7. 개정

본 방침이 변경되면 이 페이지에 개정일과 함께 게시합니다.
