RTSP Camera 개인정보 처리방침

최종 개정일: 2026-06-06 · 시행일: 2026-06-06

요약: 이 앱은 사용자 데이터를 수집하지 않습니다. 카메라·마이크 영상/음성은 사용자가 지정한 대상(같은 네트워크의 RTSP 클라이언트 또는 사용자가 직접 준비한 릴레이 서버)으로만 직접 전송되며, 개발자 서버를 거치거나 저장되지 않습니다.

1. 수집하는 개인정보

개발자는 어떠한 개인정보도 수집·저장하지 않습니다. 본 앱에는 회원가입·로그인·계정이 없습니다.

2. 카메라 및 마이크

본 앱은 영상 송출을 위해 카메라와 마이크에 접근합니다. 캡처된 영상(H.264)·음성(AAC)은 다음 대상으로 직접 전송됩니다.

  • 로컬 RTSP 모드: 같은 네트워크의 사용자 지정 클라이언트(VLC, ffplay, NVR 등)가 직접 접속하여 수신합니다.
  • 셀룰러/원격 모드(선택): 사용자가 직접 입력·준비한 제3자 릴레이 서버로 전송됩니다(BYO 모델). 해당 릴레이의 운영 주체·데이터 보관·접근통제·관할은 사용자 책임이며 개발자의 통제 밖입니다.

어느 경우에도 영상/음성은 개발자 서버를 경유하거나 저장되지 않습니다.

3. 기기 로컬 저장 항목

아래 설정값은 사용자 기기에만 저장되며 외부로 전송되지 않습니다.

  • UserDefaults: 인증 사용자명·Realm, 릴레이(RTMP) 주소 등 앱 설정.
  • Keychain: RTSP 인증 비밀번호, 릴레이 스트림 키(암호화 저장).

4. 제3자 SDK 및 추적

  • HaishinKit: 선택적 RTMP 송출 라이브러리. 사용자 데이터를 수집하지 않습니다.
  • 광고 식별자(IDFA)·SKAdNetwork·분석 SDK를 사용하지 않으며, 어떠한 추적도 수행하지 않습니다.

5. 권한

  • 카메라 — 영상 송출
  • 마이크 — 음성 송출
  • 로컬 네트워크 — 같은 네트워크 기기의 RTSP 접속 허용

각 권한은 iOS 설정에서 언제든 철회할 수 있습니다.

6. 아동의 개인정보

본 앱은 아동을 대상으로 하지 않으며, 아동으로부터 어떠한 정보도 수집하지 않습니다.

7. 개정

본 방침이 변경되면 이 페이지에 개정일과 함께 게시합니다.
